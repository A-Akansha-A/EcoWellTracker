# EcoWell Tracker 

EcoWell Tracker is a web-based wellbeing and eco-habit tracking platform.

## Features (Planned)
- Daily water intake tracking
- Step goal tracking
- Mood tracking
- Eco-friendly habit tracking
- Calendar-based daily summary
- Progressive Web App(PWA)

## Tech Stack (Planned)
- HTML, CSS, JavaScript
- Firebase(Auth, Firestore, Hosting)
- Google Technologies
